THIS PROJECT ASSUEMES THE FOLLOWING
-That you have chrome installed
-That you have a chrome driver has been installed. If you don't have it, go to https://chromedriver.chromium.org/downloads, get the one specific to your OS, and put it in the \Aumni_Selenium_Test\src\test\resources\drivers folder.
-Java is also assumed to be installed. This project is using jre 1.8.0_241 in particular.
